# 디지털인문학의 길잡이: 입문 — 부록 (코드, 데이터 저장소)

이 저장소는 **『디지털인문학의 길잡이: 입문』의 부록**으로, 책의 각 장에서 다루는 내용을 직접 실습해 볼 수 있는 **예제 데이터와 Jupyter 노트북 코드**를 제공합니다. 모든 예제는 『프랑켄슈타인』, 『로미오와 줄리엣』, 셰익스피어 소네트, 버지니아 울프의 일기 등 퍼블릭 도메인 영문학 자료를 활용합니다.

> 코드·데이터 실습이 없는 장(4장, 10장)은 이 저장소에 포함되어 있지 않습니다.

## 폴더 구조

각 장 폴더(`chapter01` ~ `chapter11`)는 다음 구성을 따릅니다.

```
chapterXX/
├── XX_노트북이름.ipynb            # 본 실습 노트북 (책의 예제 데이터 사용)
├── XX_노트북이름_practice.ipynb   # 연습용 노트북 (자신의 데이터를 업로드하여 실습)
├── data/    # 해당 장에서 다루는 예시 데이터 (노트북 없이 데이터만 제공되는 장 포함)
├── input/   # 노트북이 입력으로 사용하는 데이터
└── output/  # 노트북을 실행했을 때 생성되는 결과물의 예시
```

- **`input/`** — 노트북 실행에 필요한 입력 데이터입니다. 노트북이 GitHub에서 자동으로 내려받도록 작성되어 있으므로, 별도로 내려받지 않아도 실행할 수 있습니다.
- **`output/`** — 노트북을 끝까지 실행하면 생성되는 결과물(CSV, 그래프 이미지, HTML 시각화, 동영상 등)의 예시입니다. 자신이 실행한 결과가 올바른지 비교·확인하는 용도로 활용하세요.
- **`*_practice.ipynb`** — 본 노트북과 동일한 분석 과정을 **자신의 텍스트 파일**에 적용해 볼 수 있도록 파일 업로드 기능을 추가한 연습용 노트북입니다.

## 실행 방법

각 노트북 상단의 **"Open in Colab" 배지**를 클릭하면 별도 설치 없이 브라우저(Google Colab)에서 바로 실행할 수 있습니다. 필요한 라이브러리 설치와 데이터 다운로드가 노트북 안에 모두 포함되어 있습니다.

## 활용 안내

- 책을 읽으며 해당 장의 노트북을 함께 실행하면, 본문에서 설명한 방법이 실제 데이터에 어떻게 적용되는지 단계별로 확인할 수 있습니다.
- `_practice` 노트북에 자신의 연구 자료(소설, 일기, 희곡 등 텍스트 파일)를 업로드하면 동일한 분석을 곧바로 자신의 데이터에 적용해 볼 수 있습니다.
- 1~5장의 데이터(TEI XML, 가제티어, 네트워크 CSV)는 Gephi, QGIS, 텍스트 에디터 등 외부 도구 실습 자료로도 활용할 수 있습니다.
